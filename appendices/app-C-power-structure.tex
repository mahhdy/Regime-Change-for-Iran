\section{بیت رهبری}

مرکز اصلی قدرت که شامل:
\begin{itemize}
    \item دفتر رهبری
    \item مشاوران ارشد
    \item نمایندگان ولی فقیه در نهادها
    \item شبکه‌های مالی (ستاد اجرایی فرمان امام)
\end{itemize}
